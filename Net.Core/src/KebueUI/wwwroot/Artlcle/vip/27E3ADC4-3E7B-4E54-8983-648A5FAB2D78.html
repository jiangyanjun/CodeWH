创建dblink
create database link dbl_test connect to test identified by oracle using 'lwl';
dbl_test:dblink的名字
test:要连接的数据库的用户名
oracle:要连接的数据库的密码
lwl:本地建立的连接到远程数据库的服务名
SQL> select * from dual@dbl_test;
如果返回结果如下就说明建立成功了
