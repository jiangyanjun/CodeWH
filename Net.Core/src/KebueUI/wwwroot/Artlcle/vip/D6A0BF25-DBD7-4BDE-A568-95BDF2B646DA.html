SQL 2012 启动出现 Cannot find one or more

                        
                            
在Sql Server 2012程序，也就是SSMS 2O12启动时出现Cannot find one or more components.Please reinstall the application.
方法/步骤
  1. 1
    在运行中输入regedit.exe，启动注册表工具
  2. 2
    找到注册表中的此处路径“HKEY_CURRENT_USER\Software\Microsoft\SQL Server Management Studio”
  3. 3
    删除11.0_Config文件夹
  4. 4
    重新启动sql server 2012，也就是SSMS 2012
