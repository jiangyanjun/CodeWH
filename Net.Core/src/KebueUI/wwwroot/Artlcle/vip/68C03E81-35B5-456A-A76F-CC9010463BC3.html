不安装Oracle客户端也能使用PL/SQL
解压缩 instantclient_12_1 到 D:\Oracle\instantclient_12_1
在文件夹内建立目录, /NETWORK/ADMIN
在该目录下，新建文件tnsnames.ora
adamdb =    (DESCRIPTION =      (ADDRESS_LIST =        (ADDRESS = (PROTOCOL = TCP)(HOST = 192.168.1.7)(PORT = 1521))      )      (CONNECT_DATA =        (SID = adamdb)      )    )
添加环境变量
【ORACLE_HOME】  D:\Oracle\instantclient_12_1    【TNS_ADMIN】  D:\Oracle\instantclient_12_1\NETWORK\ADMIN    【NLS_LANG】  AMERICAN_AMERICA.UTF8
启动PL/SQL Developer，在登录窗口界面,点击取消按钮就可以进行主界面,点击Tools->Preferences，在Connection中需要配置如下两个参数
Oracle Home：D:\Oracle\instantclient_12_1
OCI Library：D:\Oracle\instantclient_12_1\oci.dll
至此配置完成,现在就可以正常使用pl/sql developer了
